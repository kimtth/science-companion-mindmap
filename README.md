# 🧪🔬🧬 Science Companion Mindmaps

> Comprehensive visual learning guides for college-level science and mathematics

[![Mermaid](https://img.shields.io/badge/Made%20with-Mermaid-ff69b4)](https://mermaid.js.org/)
[![College Level](https://img.shields.io/badge/Level-College-blue)](https://github.com)

Interactive Mermaid flowchart mindmaps covering 2,600+ concepts across physics, biology, chemistry, and mathematics. Inspired by [Grokipedia](https://grokipedia.com/).

> **Goal:** To evaluate LLMs’ world knowledge, understanding, and performance across multiple disciplines—including science, mathematics, computer science, economics, history, philosophy, and psychology.

## 📂 Structure

```
science-companion-mindmap/
├── biology.md / chemistry.md / math.md / physics.md / knowledge.md   # Navigation hubs
├── biology/          # 6 topic diagrams (green theme)
├── chemistry/        # 9 topic diagrams (orange theme)
├── math/             # 6 topic diagrams (purple theme)
├── physics/          # 6 topic diagrams (blue theme)
├── note/             # Study guides, formulas, terminology
├── knowledge/        # Extended subjects (CS, Economics, History, Philosophy, Psychology)
└── translation/kr/   # Korean translations
```

## 🚀 How to Use

- **GitHub** — Diagrams render natively in `.md` files
- **VS Code** — Install [Markdown Preview Mermaid Support](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-mermaid), press `Ctrl+Shift+V`
- **Mermaid Live** — Copy/paste to [mermaid.live](https://mermaid.live/)
- **Navigation** — Start from hub files (`biology.md`, `chemistry.md`, `math.md`, `physics.md`, `knowledge.md`)
- **Study Notes** — Formulas and equations in `note/`
- **Extended Knowledge** — Notes, terminology, and equations for CS, Economics, History, Philosophy, and Psychology in `knowledge/`
